01 / about

Who I Am

I'm Wakamiya Naufal, a cybersecurity enthusiast focused on understanding both the offensive and defensive sides of security. My interest isn't in the aesthetics of hacking,it's in the logic, the systems, and the problem-solving.

I started by learning Linux and programming, then gradually moved into penetration testing, reverse engineering, and network security. Platforms like Hack The Box and TryHackMe have been core to my hands-on learning, alongside reading documentation, CVEs, and real-world incident reports.

Outside of technical work, I have a genuine interest in the Japanese language,studying it seriously alongside my security work. I believe discipline in language learning and discipline in security research come from the same mindset: patience, consistency, and attention to detail.

I'm building toward a career in security where I can contribute meaningfully,whether that's in red teaming, threat detection, or security engineering.

03 / journey

The Journey

2022

Started Programming

First steps into coding with Python and some basic knowledge about Web Development. Learned the basics of logic, automation, and problem-solving through small projects and exercises.

2024

Moved to Linux

Switched primary environment to Linux. Learned the shell, file system, permissions, and began appreciating the control it offers.

2025

Discovered Cybersecurity

Realized security was the direction I wanted to pursue seriously. Started reading CVEs, security blogs, and understanding how vulnerabilities work.

2025

Started TryHackMe

Began structured learning on TryHackMe. Covered networking, web exploitation, Linux privilege escalation, and introductory penetration testing paths.

2025

Started Hack The Box

Joined HTB Academy and began working through more challenging boxes. Developed a methodical approach to enumeration and exploitation.

2026

Smart Parking System Project

Built a hardware integrated Smart Parking System project combining programming and embedded systems,sharpened problem solving and project planning skills.

2026

Reverse Engineering & CTFs

Dived into reverse engineering using Ghidra and x64dbg. Started participating in CTF competitions, focusing on binary exploitation and forensics challenges.

2026

NightWatch EDR,Blue Team Project

Built a proof of concept Endpoint Detection & Response tool, exploring Windows internals, event logging, and real-time threat monitoring.

2026

Cloud Security

Expanded into cloud security. Learning AWS attack paths, IAM misconfigurations, and cloud-native defensive strategies.

Now

Continuing the Path

practicing more, actively building, studying, and documenting everything on my social media.
